# Binario, hex y decimal

Este ejercicio practica la notación de trabajo del nivel: pasar entre binario, hexadecimal y decimal sin romper el agrupamiento por nibbles y bytes.

> Laboratorio: alterná entre vista binaria y vista hexadecimal sobre el mismo byte. Usá el agrupamiento en bloques de `4` bits para no convertir a ciegas.

## Consigna

Completá la tabla.

| Binario | Hex | Decimal unsigned |
|---|---|---|
| `00010000` | `?` | `?` |
| `00101100` | `?` | `?` |
| `01111111` | `?` | `?` |
| `10100111` | `?` | `?` |
| `11111111` | `?` | `?` |
| `?` | `0x80` | `?` |
| `?` | `0x3F` | `?` |

La pregunta no es solo traducir símbolos, sino conservar visible cómo cada dígito hexadecimal resume exactamente un nibble del patrón binario.
