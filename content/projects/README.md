# content/projects/

Track práctico de Forja, dividido en dos ramas:

- `focused/`: proyectos pequeños y focalizados.
- `integrating/`: proyectos integradores y multi-fase.

`project.yaml` es la fuente de verdad estructural de cada proyecto. Para reglas de naming y estados (placeholder vs authoría real), ver [CONVENTIONS.md](../../CONVENTIONS.md). Para el catálogo curado, ver [docs/forja-proyectos.md](../../docs/forja-proyectos.md).
